# Kerio Valley — Discover the Heart of the Rift

A responsive static website that showcases the raw, breathtaking beauty of Kerio Valley in Kenya's Rift Valley region. From the dramatic escarpments of Iten to the iconic Chebloch Bridge spanning the Kerio River, this site is a digital window into one of Kenya's most stunning landscapes.

## Tech Stack

- **HTML5** — semantic, accessible markup
- **Bootstrap 5.3** — responsive layout, navbar, cards, forms (CDN, no build step)
- **CSS3** (`style.css`) — custom theme colors, flexbox gallery, hero overlay
- **Vanilla JS** (`main.js`) — active nav highlighting, form feedback, smooth UX
